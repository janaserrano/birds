# Birds: Biases in Global Bird Conservation Assessments

## Overview

This repository contains code and analyses from Chapter 1 of my PhD in Ecology and Conservation Biology (McGill University), which examined how species' ecological traits, threats, and data availability influence conservation status assessments for birds worldwide.

Using global biodiversity datasets, this project integrates species occurrence, ecological traits, threat information, and conservation assessments to identify sources of bias in evaluating extinction risk and conservation status.

## Objectives

- Identify biological and data-related factors associated with bird conservation assessments.
- Explore patterns of bias in conservation status across thousands of bird species.
- Evaluate how species traits, threats, and knowledge gaps influence conservation decisions.
- Generate reproducible biodiversity data products to support large-scale conservation planning and monitoring.

## Data Sources

This project integrates multiple large-scale biodiversity datasets, including:

- IUCN Red List assessments
- Bird trait databases
- Global threat datasets
- Species occurrence and distribution data
- eBird-derived products and taxonomic resources

## Methods

- Data integration and harmonization across global biodiversity databases
- Statistical modeling in R
- Reproducible analytical workflows
- Large-scale biodiversity analyses
- Data visualization and communication of conservation-relevant patterns
